# Actividad Integradora: TypeScript y JavaScript 🚀

Este repositorio contiene dos actividades principales desarrolladas como parte del aprendizaje de programación web: un cuestionario teórico sobre **TypeScript** y un conjunto de **10 problemas prácticos resueltos en JavaScript**.

## 📖 Contenido del Proyecto

### 1. Cuestionario de TypeScript (`index.html`)
Una página principal que incluye conceptos fundamentales sobre TypeScript, organizada en tarjetas (cards):
- ¿Qué es TypeScript y sus ventajas frente a JavaScript puro?
- Instalación mediante `npm` y compilación en VS Code.
- Definición de tipos de datos, tipos literales, uniones y alias.
- Creación y utilidad de los Enums.
- Parámetros en funciones (opcionales y predeterminados).
- Mejores prácticas para mantener un código limpio y escalable.

### 2. Ejercicios Prácticos de JavaScript (`solucion1.html` a `solucion10.html`)
Se incluyen 10 páginas con una interfaz gráfica amigable, donde cada una resuelve un problema lógico distinto utilizando JavaScript puro. A los ejercicios se accede fácilmente desde los botones ubicados al final del `index.html`:
1. Calcular el **promedio de tres números**.
2. Calcular el **área de un triángulo**.
3. Determinar si un número es **par o impar**.
4. Encontrar el **mayor y menor** de tres números.
5. Calcular la **potencia** de un número.
6. **Invertir un texto** ingresado.
7. Calcular el **factorial** de un número.
8. Indicar si un valor es un **Número Perfecto**.
9. Verificar si un valor es un **Número Primo**.
10. **Contar las vocales** existentes en un texto.

## 🛠️ Tecnologías Utilizadas
- **HTML5:** Estructuración semántica de las páginas.
- **CSS3 / Bootstrap:** Diseño responsivo y moderno, utilizando componentes de Bootstrap y estilos personalizados.
- **JavaScript (Vanilla):** Toda la lógica de negocio y manipulación del DOM se encuentra de manera modular dentro de la carpeta `js/`.
