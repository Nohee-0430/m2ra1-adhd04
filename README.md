# Actividad: Animaciones en CSS 🚀
**Estudiante:** Nohelia Barillas  
**Proyecto:** Guía Teórica y Laboratorio Práctico sobre Movimiento Web.

## 🛠️ Tecnologías y Lenguajes Utilizados

El proyecto está organizado de forma modular, separando la estructura, el diseño y la lógica:

### 1. HTML5 (Estructura)
Se utilizó para definir el esqueleto de las dos páginas principales:
*   **index.html**: Contiene la base teórica organizada en tarjetas informativas.
*   **page1.html**: Estructura del laboratorio práctico con secciones divididas para ejercicios de transiciones, transformaciones y animaciones.

### 2. CSS3 (Diseño y Animación)
Es el corazón del proyecto. Se encuentra centralizado en `css/style.css`:
*   **Layout**: Se implementó **CSS Grid** y **Flexbox** para crear un diseño responsivo y moderno.
*   **Estética**: Uso de degradados, sombras suaves, tipografía personalizada (Outfit) y un diseño de tarjetas "premium".
*   **Movimiento**: Se programaron todas las **Transiciones**, **Transformaciones** (rotate, scale, translate) y **Animaciones** complejas mediante el uso de `@keyframes`.

### 3. JavaScript (Interactividad)
Ubicado en `js/main.js`, se encarga de añadir dinamismo a la experiencia del usuario:
*   **Interactividad**: Lógica para el volteado 3D de las tarjetas y cambios de estado en elementos mediante eventos de clic.
*   **Control**: Gestión de la carga del DOM para asegurar que los elementos estén listos antes de ejecutar las funciones.

### 4. Bootstrap 5 (Framework)
Se integró como apoyo para:
*   **Componentes**: Uso de modales para mostrar código y la estructura base de las tarjetas (`cards`).
*   **Responsividad**: Asegurar que el sitio funcione correctamente en dispositivos móviles.
